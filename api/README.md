## Introduction

The API for ATS project

## Project setup

```bash
# Install packages
$ npm install

# Generates SQL migration files for your declarative data model definition in the Prisma schema
$ npx prisma migrate dev

# Generates assets like Prisma Client based on the generator and data model
$ npm prisma generate
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
